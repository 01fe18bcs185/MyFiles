1. Choose any one programming language to write your code with. (C++, Java or Python)

2. If you choose C, then:
    2.1. Change directory to 'C'
    2.2. Write your code in CSolution.c file
    2.3. To validate all testcases - Run 'python test_c.py' from same directory 

3. If you choose C++, then:
    3.1. Change directory to 'CPP'
    3.2. Write your code in CppSolution.cpp file
    3.3. To validate all testcases - Run 'python test_cpp.py' from same directory

4. If you choose Java, then:
    4.1. Change directory to 'Java'
    4.2. Write your code in JavaSolution.java file
    4.3. To validate all testcases - Run 'python test_java.py' from same directory

5. If you choose Python, then:
    5.1. Change directory to 'Python'
    5.2. Write your code in PythonSolution.py file
    5.3. To validate all testcases - Run 'python test_python.py' from same directory


POSSIBLE ERRORS AND THEIR RESOLUTION:
1. If 'python' command doesn't work, try issuing with 'python3' instead

2. C/C++ programs need GCC and G++ compilers. If not installed, use: 
   "sudo apt install gcc g++"

3. Java programs need JDK and JRE. If not installed, use:
   "sudo apt install default-jdk"
   "sudo apt install default-jre"
